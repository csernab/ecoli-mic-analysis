# Antimicrobial resistance in Spanish livestock *E. coli*

This repository contains the anonymized data and R scripts used to analyse
phenotypic antimicrobial resistance in indicator and third-generation
cephalosporin-resistant (3GCR) *Escherichia coli* from Spanish livestock,
2014-2023.

## Files

- `data/ecoli_mic.csv`: MIC and WT/NWT data.
- `data/cluster_probabilities.csv`: cluster assignments and posterior probabilities.
- `data/antimicrobial_sales.csv`: national veterinary antimicrobial sales.
- `data/ecoffs.csv`: epidemiological cut-off values used in the analysis.
- `data/model_selection.csv`: saved mixture-model selection results.
- `scripts/`: R scripts used in the analysis.
